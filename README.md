## Overview
This project performs Exploratory Data Analysis (EDA) on Zomato restaurant data to identify patterns and insights related to ratings, cuisines, and pricing across different cities.

## Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Excel
- Power BI

## Key Steps
1. Data Cleaning: Removed duplicates and handled missing values in the dataset.  
2. Exploratory Analysis: Analyzed top cuisines, restaurant ratings, and location-based trends.  
3. Visualization: Created interactive dashboards using Power BI to represent key insights visually.

## Outcome
- Identified popular cuisines in different cities.  
- Explored relationship between restaurant pricing and ratings.  
- Created dashboards that support business decisions with clear visual insights.
